### After Completing This Week

You should be able to or know:

-   Write strongly typed and scalable code (TypeScript).
-   Define and enforce types, interfaces, enums, and classes.
-   Use generics and type inference to make reusable, type-safe utilities.
-   Organize code with modules, imports, and exports.
-   Build object-oriented applications with proper access control (public, private, protected).
-   Understand how NodeJS works under the hood.
-   Use core Node.js modules like http, fs, path, and events to build low-level functionality.
-   Create a server from scratch using the http module (no frameworks).
-   Handle asynchronous programming using callbacks, and async/await.
-   Use EventEmitter to implement event-driven patterns.
-   Manage files and directories using the fs module (e.g., read/write logs or JSON files).
-   Build RESTful APIs with ExpressJS.
-   Define routes for GET, POST, PUT, DELETE requests.
