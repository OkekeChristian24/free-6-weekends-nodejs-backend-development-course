### Tools Installed in This Session

During this session, we installed and configured the essential tools required for backend development with **Node.js**.

| Tool                             | Purpose / Description                                                                                          |
| -------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **Node.js**                      | JavaScript **runtime environment** used to execute JS code outside the browser.                                |
| **Visual Studio Code (VS Code)** | A powerful and lightweight **code editor** for writing, debugging, and managing your Node.js projects.         |
| **MySQL Server & Workbench**     | **Relational database system** and GUI client for managing schemas, tables, and queries.                       |
| **Docker**                       | **Containerization software** that lets you package and run your applications in isolated environments.        |
| **Git + GitHub**                 | **Version control system** for tracking code changes and collaborating. Includes creating your GitHub account. |
| **Postman**                      | A popular **HTTP client** for testing REST APIs, sending requests, and viewing responses.                      |
